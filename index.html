What's this?

This is timeer, a little checkpoint app I wrote in JavaScript to help track my progress running.

Key Technologies used: LocalStorage, JSON, jQuery, Twitter Bootstrap, UnderscoreJS, Highcharts

You can hack the code here or fork it on Github

Todos:
  • Show Time on y-axis of chart as 00:00.00
  • Show how much data is being used in Localstorage vs. what is available
  • Save Localstorage data
  • Clear LocalStorage
